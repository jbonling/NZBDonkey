Search Engines Settings

Active: check if you want to use this search engine
Search Engine Name: name of the search engine
Search URL: URL called for a NZB file search. Use "%s" for the search string.
Response Type: Type of response sent back from the search engine. Can be either a HTML page or a JSON object.
JSON object path: If response type is JSON, this is the JSON object path for the NZB file ID (divided by periods). Example: { results: [ { nzbid: 123456 }, ... ], ... } => "results.0.nzbid"
Regex expression: If response type is HTML, this is the regex expression to be used to search in the HTML search results page for the NZB file ID. The NZB file ID has to be in a capturing group ().
Group no.: number of the capturing group in the regex expression that does contain the NZB file ID.
Download URL: URL called to download the NZB file. Use "%s" for the NZB file ID.

Interception Settings

Active: check if you want to intercept NZB file downloads on this domain
Domain: base domain where you want to intercept NZB file download. CAUTION: do not add "www" or any other subdomains.
Handling of Form Data: choose from the options how POST Form Data should be handled for this domain.
If the default value is resulting in constant error messages regarding invalid or empty NZB files, try one of the other options.
